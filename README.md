# WhoIntel
EVE online secure tool

Замена устаревшего vintel.

https://whointel.space/

## Разработка
Установите зависимости
```
yarn install
```

Запустите dev сервер
```
yarn electron:serve
```

Сборка дистрибутива
```
yarn electron:build
```
